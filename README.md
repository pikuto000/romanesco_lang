# [開発中] Romanesco プログラミング言語

Romanesco（ロマネスコ）は、型に自由な制約をかけられる **静的制約** 型付け言語であり、究極の疎結合性を追求したメタプログラミング言語です。

### 特徴

- **強い同図像性**: 適用（Apply）そのものをデータとして扱え、プログラムとデータの境界がありません。
- **第一級パーサー**: 関数自身が「自分の引数をどうパースするか」を決定できます。
- **動的自己進化**: 実行中にトークン分割ルール（レキサー）を書き換え、即座に自身を読み直す「反復パース」を備えています。
- **盤石な衛生性**: 依存関係グラフに基づき、マクロによる意図しない名前衝突を完全に防ぎます。
- **環境の抽象化**: マクロや構文規則といった概念さえも、環境（シンボルテーブル）上のプロパティとして扱われます。
