// 1. 静的な事前計算マクロ (ラムダなし)
comptime calc-static (+ 10 20)

write calc-static

// 2. 静的なAST変換マクロ (ラムダなし)
macro-ast hello-macro (make-atom HelloMacro)

write hello-macro

// 3. 既存の動的マクロ (引数を使う場合はラムダが必要)
macro-ast id-macro \ args (get-arg 0) _

write id-macro (+ 1 2)

// 4. 通常の定義
:- secret _ 42
write secret
