Romanesco 証明エンジン 開発ロードマップ

- [x] SolveTree による投機的並列探索 (DeepStep)
- [x] グローバル補題キャッシュ (Lemma Cache) と失敗キャッシュの実装
- [x] 構造的パターン繰り返し検知による循環回避
- [x] 拡張デバッグロガー (スレッド識別、タイムスタンプ、処理時間計測)

- [x] 共帰納法の安定化: ストリームの自己シミュレーション (`bisim`) の高速解決達成

2. 短期的な課題 (Priority: High) ⏳
- [ ] 依存型の高度な定理: `∀n. ∀v:Vec(A, n). vmap(id, v) = v` のような、型と値が密接に絡む定理の検証
- [ ] モナド則の検証: `bind(return(x), f) = f(x)` 等のモナド三則を Rewriter と Tactic 両面から強化
- [ ] 補題の自動発見 (Proof Mining): 探索中に頻出する中間形式を自動的に補題として抽出する仕組みの検討

3. 中長期的な研究・開発項目 (Priority: Medium) 🚀

- [ ] 高次帰納型 (HIT) の深化: 円周 $S^1$ 以外の、より複雑な形状（商集合、基本群など）への応用
- [ ] 分離論理のフレーム自動生成: `P * ?Frame ⊸ Q` を解く際の `?Frame` の自動推論

4. 目標 (Vision) 🎯
- [ ] あらゆる論理体系 (分離、線形、時相、HoTT) がシームレスに協力し、依存型を持つプログラムの性質を自動証明できる、世界で最も柔軟な証明エンジンを目指す。
